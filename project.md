1. Project Title:
   
        Personal Expense Tracker System

3. Problem Statement:
   
           Managing daily expenses manually is often difficult, leading to a lack of financial awareness.
             Users often lose track of their spending,making it hard to save money or analyze their financial habits.
              There is a need for a digital, easy-to-use platform that records and displays expenses in real-time.

 4. Project Objective:
    
         # To create a user-friendly interface for recording daily expenses.

         # To store financial data securely in a database (MySQL).

         # To provide instant visibility of spending history.

          # To automate expense tracking, reducing the need for manual calculations.

  4. Module List:
     
         # User Interface Module: Handles the input form for entering expense details (Title and Amount).

         # Data Processing Module (Backend): Manages the business logic and API endpoints using Spring Boot.

         # Database Management Module: Handles data persistence, retrieval, and updates in MySQL.

         # History/View Module: Displays the list of all recorded expenses on the dashboard dynamically.

5. Table List
6.         Database Name: my_expense_db
7.         Column Name           Data Type              Description
8.         id                    BIGINT             Primary Key, Auto-increment
9.         title                 VARCHAR(255)       Name of the expense
10.        amount                DOUBLE             Cost of the expense

6. Expected Outcome:

        The application provides a clean, functional dashboard where users can efficiently log their expenses.
        It eliminates the need for manual record-keeping, ensuring that users have a clear, accurate, and
        digital history of their spending to help make better financial decisions.
